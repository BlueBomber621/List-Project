Cullinary Product Addition App

created with HTML, CSS, and JS

This is a customer input system.
Using this you can add any items to the list, even unrealated ones.


This project will take what you type in the text inputs and add it to a list below. You can add as many items as you would like into the list. You can also optionally add a description to the item, and it will include that with the list item.


This project will include ways to display a message when you submit an empty input, and clear the input once subbmitted.
